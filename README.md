# Pandor-AI — Landing Futurista (Vite + React)

Proyecto demo: landing page futurista llamada **Pandor-AI**.

Ruta del proyecto (obligatoria): `C:\Users\HELL\Desktop\boletin-pandora-ia`

Instrucciones rápidas (PowerShell):

```powershell
# Crear la carpeta principal exactamente en la ruta solicitada
mkdir "C:\Users\HELL\Desktop\boletin-pandora-ia"; cd "C:\Users\HELL\Desktop\boletin-pandora-ia"

# Inicialmente ya se han creado los archivos en esta ruta por el asistente.
# Instalar dependencias (usa npm o pnpm/yarn según prefieras)
npm install

# Iniciar servidor de desarrollo
npm run dev
```

Dependencias principales incluidas en `package.json`:
- `react`, `react-dom`
- `framer-motion` para animaciones suaves y scroll
- `tsparticles` + `react-tsparticles` para partículas y efectos de fondo
- `vite` como bundler

Notas:
- La tipografía principal se importa desde Google Fonts (`Orbitron` y `Inter`).
- Los componentes están en `src/components`.
- `src/styles/globals.css` contiene las variables de color, estilos neon y glassmorphism.

Siguientes pasos sugeridos:
- Abrir el proyecto en VS Code.
- Ejecutar `npm install` y luego `npm run dev`.
- Reemplazar imágenes en `src/assets` según desees.

Si quieres que haga el commit Git o que añada imágenes adicionales, dímelo y lo hago.
